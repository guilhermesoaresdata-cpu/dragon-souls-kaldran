# Mares

## Visao Geral

Registro dos mares oficiais de Kaldran.

## Informacoes Oficiais

- Asgardia e separada por mar.
- Pelo mar e impossivel acessar Zharveth.
- A abertura do estreito entre Utgard e Zharveth depende do mar.
- Existe o Mar de Gelo.
- O Mar de Gelo fica entre Vintergard, Utgard e Asgardia no mapa conhecido.
- Existe o Oceano Infinito.
- O Oceano Infinito fica a leste de Kaldran no mapa conhecido.

## Detalhes

O mapa conhecido mostra rotas maritimas atravessando os mares entre os continentes.
