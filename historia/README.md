# Historia

## Visao Geral

Registro dos eventos historicos oficiais de Kaldran.

## Informacoes Oficiais

- Kaldran ja foi um supercontinente quebrado.
- A Guerra da Ruptura causou a quebra do supercontinente.
- A Porta do Inferno surgiu em Zharveth durante a invasao demonica.
- Os Valerianos sobrecarregaram suas Runas para reescrever a geografia de Kaldran.
