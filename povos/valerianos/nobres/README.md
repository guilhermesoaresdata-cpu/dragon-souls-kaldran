# Valerianos Nobres

## Visao Geral

Registro dos Valerianos Nobres conhecidos em Kaldran.

## Informacoes Oficiais

- [Solaris](solaris.md) e um Valeriano Nobre da Primeira Geracao.
- Valerianos Nobres tambem sao chamados de Primeira Geracao.
- Sao rarissimos.
- Possuem sangue mais puro.
- Possuem matrizes runicas mais primitivas.
- Possuem poder devastador.
- Possuem longevidade extrema.
- Possuem excentricidade perigosa.
- Podem manipular o Imperio Sagrado em Asgardia.
- Podem viver isolados em Vintergard ou Zharveth.
- Sao os mais proximos do que os povos chamam de deuses.

## Detalhes

Valerianos Nobres carregam matrizes runicas especialmente puras, primitivas e devastadoras.

Para os povos de Kaldran, estes sao os mais proximos dos deuses. Para si mesmos, talvez sejam apenas sobreviventes velhos demais para suportar a propria memoria.
